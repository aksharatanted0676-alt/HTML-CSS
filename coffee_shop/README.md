# Coffee Shop Project

☕ Mumbai Brew Café

A modern and aesthetic café website inspired by Mumbai’s vibrant night culture.
This project showcases a premium coffee shop experience with smooth animations, steam effects, gallery hover zoom, and embedded Google Maps.

🌆 About The Project

Mumbai Brew Café is a responsive coffee shop website built using pure HTML and CSS.
It highlights handcrafted coffee, creamy milkshakes, and cozy café vibes across Mumbai locations.

The design focuses on:
Warm dark theme ☕,
Smooth animations ✨
Interactive hover effects 🎨,
Steam animation on coffee image 🌫,
Mumbai-themed locations 📍


✨ Features

🎨 Modern dark café aesthetic

☕ Animated steam effect over coffee image

🖼 Image gallery with hover zoom effect

📍 Mumbai location cards with glassmorphism effect

🗺 Embedded Google Map of Mumbai

📱 Fully responsive design

📩 Newsletter subscription section

🧭 Clean navigation bar


🛠 Technologies Used:
HTML5,
CSS3,
Flexbox,
CSS Grid,
CSS Animations,
Google Fonts (Poppins),
Embedded Google Maps
