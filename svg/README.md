# SVG Project

SVG Shapes & Icons Collection

A creative collection of SVG graphics and path-based designs built using pure HTML and SVG.
This project demonstrates how to create shapes, curves, and custom icons using SVG elements like path, circle, rect, ellipse, and line.


✨ Project Overview

This project includes:
➖ Straight Line (using path),
▭ Rectangle (using path),
🔺 Triangle (using path),
🌊 Quadratic Curve (using Q command),
🤖 Android-style SVG Logo (custom shapes),
🔴 Pokéball-style Design (using circles, arcs & layers).

It is a beginner-friendly SVG practice project for understanding vector graphics.

🛠 Technologies Used:
HTML5,
SVG (Scalable Vector Graphics),
SVG Path Commands,
Basic Styling
