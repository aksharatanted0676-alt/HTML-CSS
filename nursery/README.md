# Nursery Project

🌿 GreenLeaf Plant Nursery Website

A responsive and modern Plant Nursery Website built using HTML5 and CSS3.
This project showcases a clean green-themed layout for a plant nursery business including a hero section, plant cards, about section, and contact form.

✨ Features

🌱 Nature-inspired green design

🏡 Full-screen Hero section with background image

🌿 Plant product cards with hover animation

📖 About Us section

📩 Contact form UI

📱 Fully responsive layout

🎨 Smooth hover effects


🛠️ Built With

HTML5

CSS3

Flexbox

CSS Grid

Media Queries
