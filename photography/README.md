# Photography Project

📷 Delphine Rose – Photography Portfolio Website

A modern, elegant photography portfolio website built using HTML5 and CSS3.
This project showcases a luxury-style layout with editorial typography, soft neutral tones, layered images, and a refined portfolio section — perfect for photographers and creative professionals.


✨ Features

🎨 Minimal & Elegant Design

🖼️ Full-screen Hero Section with overlay

📰 Editorial Typography (Playfair Display, Montserrat, Allura)

🧩 Grid-based About Section with layered images

📸 Portfolio Gallery with vertical labels

📱 Responsive Design (Mobile optimized)

🌿 Soft neutral aesthetic color palette


🛠️ Built With

HTML5

CSS3

Google Fonts

Flexbox

CSS Grid

Media Queries

Download or clone the repository

Open the project folder

Double click on index.html

Open in your browser

No additional setup required.
