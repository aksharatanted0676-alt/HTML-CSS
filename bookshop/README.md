# Bookshop Project

BookNest – Online Book Showcase Website
 Project Overview
BookNest is a simple and elegant book showcase website created using HTML and CSS.
It displays featured books with purchase links and highlights cozy reading corner ideas for users.
This project focuses on:
Clean layout design
Proper section structuring
Styling using CSS
External linking to book purchase pages
🎯 Features
🏠 Hero section with welcome message
📖 Featured books section
🔗 Clickable book links (opens in new tab)
🪑 Cozy reading corner showcase
🎨 Soft aesthetic color theme
📱 Simple responsive layout using Flexbox
🛠️ Technologies Used
HTML5
CSS3
Flexbox layout
External hyperlinks (Amazon book links)
