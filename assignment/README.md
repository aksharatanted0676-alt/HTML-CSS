# Assignment Project

 Notion Style Landing Page
 
A clean and modern Notion-inspired landing page built using pure HTML and CSS.
This project recreates a minimal SaaS-style homepage with smooth animations, gradient background, and responsive layout.


✨ Features

🌈 Animated gradient background

🖥 Clean and minimal UI

📱 Fully responsive layout

🚀 Floating hero illustration animation

🎯 Modern call-to-action buttons

🧭 Navigation bar with hover effects

🏷 Trusted brands section

🛠 Technologies Used:
HTML5,
CSS3,
Flexbox,
CSS Keyframe Animations,
Responsive Media Queries


📂 Project Structure

Notion-Style-Landing-Page/

│
├── index.html

├── illustration.jpg

└── README.md


🚀 How to Run

Download or clone the repository

Make sure illustration.jpg is in the same folder as index.html

Open index.html in your browser

No frameworks or libraries required.

Large bold hero typography
