# Web Project

🌐 WebDesignPro – Creative Web Design Agency Website

A modern and responsive business website template built using HTML and CSS.
This project showcases a professional web design agency layout with hero section, services, about, projects, and footer.

✨ Features

🎨 Modern gradient hero section

🧊 Glass-effect sticky navbar

🚀 Smooth hover animations

📱 Fully responsive layout

💼 Services section with animated cards

🖼 Project showcase grid

🎯 Call-to-action buttons

🌙 Professional dark footer


🛠 Technologies Used:
HTML5,
CSS3,
Flexbox,
CSS Animations,
Media Queries (Responsive Design),
Google Fonts (Poppins)


📂 Project Structure

WebDesignPro/

│
├── index.html

└── README.md

🚀 How to Run

Download or clone the repository

Keep index.html and style.css in the same folder

Open index.html in your browser

No frameworks or libraries required.
