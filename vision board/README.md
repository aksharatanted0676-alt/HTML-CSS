# Vision Board Project

🎉 Welcome 2026 – Vision Board Website ✨

An interactive and animated New Year 2026 Vision Board Website built using HTML, CSS, and JavaScript.
This project features a colorful animated background, floating emojis, background music, and a popup vision board gallery.


🌟 Features

🎨 Animated gradient background

🎆 Floating emoji animations

🎵 Background music with play button

🖼️ Vision Board popup gallery (10 image grid)

💻 Responsive centered layout

✨ Smooth popup overlay effect


🛠️ Built With

HTML5

CSS3 (Animations + Grid)

JavaScript (DOM Manipulation)

🚀 How to Run

Download or clone the repository

Make sure all images and vision.mp3 are inside the same folder

Open index.html in your browser

Click "Explore 2026"

Enjoy your animated Vision Board 🎉


🎯 Project Highlights

The main page shows:
🎉 Welcome 2026 heading,
Motivational message,
Explore button

On clicking Explore 2026:
A full-screen popup opens,
Displays a 5-column image grid,
